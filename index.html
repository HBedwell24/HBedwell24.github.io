ABOUT ME

As someone who began their education within a web development setting, I have always had a passion for user experience, as well as general application functionality. In addition, my heartfelt love for game photography/digital art has often allowed me to explore outside of my scope of traditional art styles and has served to expand my knowledge set of compatible fonts and/or color swatches.

This background, when coupled with my more recent Java proficiency coming out of my third year of college, would provide for the ideal foundation to begin learning the fundamentals of Android development and has allowed me to study the breadth of this topic at an astounding pace. Since then, I have gained considerable experience in the programming methodologies of the current age, including Agile/Scrum Development and Object Oriented Programming (OOP). In addition, I have been able to work with numerous technologies, including but not limited to: Retrofit, Gson, PHP, and the YouTube Data API Java Client, which have all been utilized in coalition with the projects (group and personal) that I have had the opportunity to work on/experience.  

Of the Android projects I have partaken in thus so far, my most impressive application utilizes the Google YouTube Data API to access trending videos on YouTube, in addition to the user’s own subscriptions, which are displayed onto a Fragment using a CardViewAdapter. The user is also able to search for videos on YouTube using the Data APIs Search:list endpoint, with recent search queries being saved to a SQLite database for reuse at another point. To personalize the in-app experience, there is also a way to clear search history by wiping the SQLite database of all queries, not to mention change the UI to Dark theme for less eye strain at night (to see more details on this project, please visit the 'Projects' section).

RECENT PROJECTS

ANDROID YOUTUBE DATA API V3 INTEGRATION

July 2019 - Present

Using the Google API’s Search:list, Videos:list, and CommentThreads:list endpoints, relevant videos from a authenticated user’s (OAuth) perspective on YouTube were pulled and displayed inside a CardViewAdapter, allowing one to view the top 25 trending videos on YouTube, in addition to their own subscription feed. Technologies used included Retrofit (for mapping the data to Java objects), Gson (for serializing and deserializing Java objects to and from JSON), the YouTube Data API Java Client, and the Material Design Color Tool, which was used to generate interest amongst key features within the various activities.

FACIAL RECOGNITION WEB ENABLED CAMERA

August 2018 - May 2019

A UVC compatible camera was triggered by a PIR motion sensor to store a .png image on an Arduino Yun. After this task had been completed, the image was then uploaded to Amazon S3 (to be compared on an image to image basis with existing records in the opposing S3 bucket). If a face was captured that did not exist in the collection, a lambda function would be triggered, which would then send a warning message to the user.
